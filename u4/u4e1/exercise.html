In this exercise you must create two different classes, which derive from the Bird class. This class is declared in bird.hh.

You must implement two new classes, Duck, in duck.hh, and Owl, in owl.hh. These classes inherit from Bird, and they implement the speak() member function.

When a owl speaks, it prints out its name and the text WHUU delimited by : to the output stream. When a duck speaks, it prints out its name and the text QUACK delimited by : to the output stream. Both messages are terminated by the newline character. For example, if a Duck called "Matti" spoke to std::cout, the text "Matti: QUACK" would be printed to the standard output.

You must also implement the class Aviary, which is used to house different kinds of birds. The header file stub aviary.hh can be downloaded from this page. You must fill it and return both the header file and its implementation in aviary.cc. The header file has more specific instructions.

We will also give you a very simple test program to test your classes.

Notes:

  • To get operator[] to work with both const and nonconst aviaries, you must actually implement it twice. The return value must be a pointer to Bird.
  • The member function addBird() takes its parameter as a pointer to prevent object slicing.
  • If somebody tries to copy an instance of Aviary, the compilation must fail. It is your job to make this happen.
